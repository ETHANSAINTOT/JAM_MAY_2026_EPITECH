# Breaking Track - FR 🇫🇷

Le projet Breaking Track est un projet réalisé durant la Jam d'EPITECH de mai 2026 sur le thème de la musique.

### Pré requis
* Python 3
* Caméra (webcam)
* Microphone
* Sortie son
* Navigateur internet
* Connexion internet active

### Lancer le projet
Dans un terminal de commande
```bash
python3 -m http.server 8000
```

Puis dans votre navigateur
```url
localhost:8000/blindtest.html
```
